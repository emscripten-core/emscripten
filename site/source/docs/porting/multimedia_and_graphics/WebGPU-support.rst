.. _WebGPU-support:

============================
Using WebGPU in Emscripten
============================

Emscripten supports using WebGPU via `Emdawnwebgpu <https://dawn.googlesource.com/dawn/+/refs/heads/main/src/emdawnwebgpu/pkg/>`_,
an external Emscripten port that implement's `Daswn's <https://dawn.googlesource.com/dawn>`_ C API on top of WebGPU.
See `This article <https://developer.chrome.com/docs/web-platform/webgpu/build-app>`_ for an example of how to use it.
