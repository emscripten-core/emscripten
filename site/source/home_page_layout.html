Emscripten is an LLVM-based project that compiles C/C++ into highly-optimizable JavaScript in asm.js format. This lets you run C/C++ on the web at near-native speed, without plugins.

Porting

Compile your existing projects written in C or C++ and run them on all modern browsers.

APIs

Emscripten converts OpenGL into WebGL, and lets you use familiar APIs like SDL, or HTML5 directly.

Fast

Thanks to LLVM optimizations and Emscripten's JavaScript-specific optimizations, code runs at near-native speed.
